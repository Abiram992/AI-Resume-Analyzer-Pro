# 🚀 AI Resume Analyzer Pro

> Next-Generation AI Resume Intelligence Platform

AI Resume Analyzer Pro is a modern AI-powered web application that analyzes resumes using ATS scoring, NLP, AI feedback, JD matching, resume rewriting, and recruiter-style analytics dashboards.

---

# 🌟 Features

## ✅ ATS Resume Scoring
- Intelligent ATS compatibility analysis
- Resume strength evaluation
- Keyword optimization

## ✅ AI Resume Feedback
- AI-generated improvement suggestions
- Professional optimization insights
- Resume enhancement recommendations

## ✅ AI Resume Rewriter
- Rewrites weak resume content professionally
- Adds ATS-friendly action verbs
- Improves resume impact

## ✅ Job Description Matching
- Compare resume with job descriptions
- JD match percentage calculation
- Hiring probability insights

## ✅ Skill Extraction using NLP
- Automatic skill detection
- Missing skills identification
- Skill categorization

## ✅ Career Role Matching
- AI-powered role fit analysis
- Full Stack Developer
- AI/ML Engineer
- Data Analyst
- Cloud Engineer

## ✅ AI Interview Question Generator
- Personalized interview preparation
- Resume-based AI questions
- Technical + HR questions

## ✅ Resume Analytics Dashboard
- ATS Gauge Meter
- Radar Charts
- Skill Analytics
- Communication Score
- Project Strength Analysis

## ✅ Downloadable PDF Reports
- AI-generated resume analysis reports
- ATS score reports
- Skill analysis reports

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Resume-Analyzer-Pro.git
```

## Move Into Folder

```bash
cd AI-Resume-Analyzer-Pro
```

## Create Virtual Environment

```bash
python3 -m venv .venv
```

## Activate Virtual Environment

### Mac/Linux

```bash
source .venv/bin/activate
```

### Windows

```bash
.venv\Scripts\activate
```

## Install Requirements

```bash
pip install -r requirements.txt
```
